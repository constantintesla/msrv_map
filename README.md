# Генератор KMZ/KML карт

Веб-приложение для создания карт с координатной сеткой, метками и экспортом в форматы KMZ и KML.

## Возможности

### Основные функции

- **Создание координатной сетки** с настраиваемым размером квадратов в метрах
- **Размещение меток** различных типов на карте с описаниями
- **Редактирование меток**: 
  - Перемещение перетаскиванием
  - Изменение описания через кнопку редактирования или Ctrl+Click
- **Выбор зоны** для генерации сетки
- **Краевые метки**: буквы слева/справа и цифры сверху/снизу с возможностью включения/выключения
- **Специальные квадраты**:
  - **A1**: отображает масштаб сетки (например, `<100 м>`) по центру
  - **A2**: содержит крест и нумерацию 1-4 для точного указания местоположения
- **Предпросмотр печати** с настройками формата (A4/A3, книжная/альбомная) и автоматическим масштабированием шрифта
- **Экспорт в KMZ и KML** форматы с сохранением всех элементов
- **Импорт KMZ/KML** файлов
- **Сохранение и загрузка проектов** с названием и датой

### Настройки карты

- **Типы подложек**:
  - Спутниковая съемка (Esri World Imagery)
  - Топографическая карта (OpenStreetMap)
  - Карта с высотами (OpenTopoMap) - показывает рельеф с высотными отметками
- **Настройка сетки**:
  - Цвет линий сетки
  - Толщина линий сетки
  - Размер квадратов (в метрах)

### Интерфейс

- Современный и интуитивный интерфейс
- Боковая панель с настройками
- Интерактивная карта с возможностью масштабирования и перетаскивания
- Режим предпросмотра печати с масштабированием

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd msrv_map
```

2. Откройте `index.html` в браузере или используйте локальный веб-сервер:
```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server -p 8000
```

3. Откройте в браузере: `http://localhost:8000`

## Использование

### Создание сетки

1. Выберите **тип карты** (спутник/топографическая)
2. Нажмите **"Выбрать зону для сетки"** и нарисуйте прямоугольную область на карте
3. Введите **размер сетки** в метрах
4. Нажмите **"Создать сетку"**

### Добавление меток

1. Выберите **тип метки** (стандартная, предупреждение, опасность, информация, КПП)
2. Введите **описание** метки (необязательно)
3. Нажмите **"Режим добавления меток"**
4. Кликните на карте, чтобы разместить метку

### Редактирование меток

- **Перемещение**: зажмите метку и перетащите в новое место на карте
- **Изменение описания**: 
  - Нажмите кнопку ✏️ в списке меток в боковой панели
  - Или используйте **Ctrl+Click** (Cmd+Click на Mac) по метке на карте
- В списке меток отображаются координаты каждой метки

### Настройка краевых меток

- Установите флажки для отображения меток на нужных сторонах:
  - **Слева** — буквы (A, B, C...)
  - **Справа** — буквы
  - **Сверху** — цифры (1, 2, 3...)
  - **Снизу** — цифры

### Экспорт

1. Нажмите **"Экспортировать KMZ"** или **"Экспортировать KML"**
2. Файл будет скачан автоматически
3. Файл содержит:
   - Все метки с описаниями и типами
   - Координатную сетку с настраиваемым цветом и толщиной линий
   - Названия квадратов (масштаб для A1)
   - Крест и цифры 1-4 в квадрате A2
   - Краевые метки (буквы и цифры по краям сетки)

**Примечание**: 
- Все текстовые элементы оптимизированы для отображения на мобильных устройствах
- Размеры шрифтов настроены для корректного отображения в Google Earth и других KML/KMZ просмотрщиках

### Импорт

1. Нажмите **"Выбрать файл"** в разделе импорта
2. Выберите файл KMZ или KML
3. Метки и сетка будут автоматически добавлены на карту

### Печать

1. Нажмите **"Предпросмотр печати"**
2. Выберите формат бумаги (A4/A3) и ориентацию (книжная/альбомная)
3. Используйте ползунок масштаба для точной настройки
4. Карта в предпросмотре перемещаема и масштабируема
5. Нажмите **"Печать"**

**Примечание**: 
- Размер шрифта автоматически масштабируется при изменении масштаба карты в предпросмотре
- Все элементы (метки, сетка, краевые метки, цифры в A2) оптимизированы для печати

### Сохранение и загрузка проектов

1. **Сохранение проекта**:
   - Создайте сетку и/или добавьте метки
   - Введите название проекта
   - Нажмите **"Сохранить проект"**
   - Проект сохраняется в браузере (localStorage)

2. **Загрузка проекта**:
   - В списке сохраненных проектов нажмите кнопку 📂
   - Все данные (метки, сетка, настройки) будут восстановлены

3. **Удаление проекта**:
   - Нажмите кнопку 🗑️ рядом с проектом
   - Проект будет удален из списка

**Примечание**: Проекты хранятся локально в браузере. При очистке данных браузера проекты будут удалены.

## Технологии

- **Leaflet.js** — библиотека для интерактивных карт
- **JSZip** — работа с ZIP архивами (для KMZ)
- **Vanilla JavaScript** — без фреймворков
- **HTML5/CSS3** — современная верстка
