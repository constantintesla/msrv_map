# msrv_map — Генератор карт с координатной сеткой

Веб-приложение для создания карт с координатной сеткой квадратов и точками интереса.
Предназначено для подготовки карт к полевой работе: экспорт на мобильные устройства
(MapsPlus, AlpineQuest, Google Earth) и печать.

## Возможности

- **Интерактивная карта** — Leaflet с 3 подложками (спутник, топо, OpenTopoMap)
- **Координатная сетка** — настраиваемый размер, цвет, толщина, начальная буква
- **Специальные ячейки** — масштаб (A1) и "улитка" (A2)
- **Краевые метки** — буквы/цифры по сторонам сетки
- **Маркеры** — 5 типов (стандартный, предупреждение, опасность, информация, КПП)
- **Сдвиг сетки** — точное позиционирование относительно объектов
- **Экспорт KMZ** — сетка как GroundOverlay для единообразного вида на всех устройствах
- **Экспорт KML** — векторная сетка, совместимость с MapsPlus/AlpineQuest
- **Экспорт PNG** — высокое разрешение (3000px), offscreen Canvas
- **Импорт KMZ/KML** — восстановление сетки и маркеров
- **Проекты** — сохранение/загрузка состояния в браузере (localStorage)
- **Конвертер** — преобразование между KML, KMZ и GPX

## Требования

- Node.js 18+
- npm

## Установка

```bash
git clone <repo-url>
cd msrv_map
npm install
```

## Запуск

### Режим разработки

```bash
npm run dev
```

Открыть http://localhost:5173 в браузере.

### Production сборка

```bash
npm run build
npm run preview
```

Статика собирается в папку `dist/`. Можно разместить на любом
статическом хостинге (Nginx, Apache, GitHub Pages, Netlify и т.д.).

### Тесты

```bash
npm test
```

## Стек технологий

| Vite | Сборка, dev-сервер |
| TypeScript | Типобезопасность |
| Leaflet | Интерактивная карта |
| JSZip | Работа с KMZ (ZIP) |
| Vitest | Unit-тесты |

## Структура проекта

```
src/
  core/       — бизнес-логика (карта, сетка, маркеры, проекты)
  export/     — экспорт/импорт (KML, KMZ, GPX, PNG)
  converter/  — конвертер файлов
  ui/         — интерфейс (sidebar, табы, модалки)
  utils/      — утилиты (geo, dom, xml)
styles/       — CSS стили
```
